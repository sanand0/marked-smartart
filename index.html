Marked.js Chevron Plugin Demo

This demo shows how to use the Chevron Plugin with Marked.js to create process flow diagrams in your markdown content.

New: The plugin now supports named parameters for both diagram-level and chevron-level styling!

Basic Example

```chevron
Planning
Development
Testing
Deployment
```

Legacy Format: Custom Width and Height

```chevron 150 80
Backlog
Sprint Planning
Daily Scrum
Sprint Review
Retrospective
```

New Format: Named Parameters

```chevron width=180 height=90 fontSize=16
Backlog
Sprint Planning
Daily Scrum
Sprint Review
Retrospective
```

Legacy Format: Custom Colors

```chevron
Idea | #9C27B0
Research | #3F51B5
Design | #03A9F4
Development | #4CAF50
Testing | #FFC107
Deployment | #FF5722
```

New Format: Per-Chevron Options

```chevron width=150 height=100
Idea | color=#9C27B0 fontSize=18
Research | color=#3F51B5 fontSize=16
Design | color=#03A9F4 fontSize=14
Development | color=#4CAF50 fontSize=12
Testing | color=#FFC107 fontSize=10
Deployment | color=#FF5722 fontSize=8
```
